Simple How-To Instructions

  1. Modify the "main" code file:
    • Java: that's the Main.java file
    • Python: that's the main.py file
  2. Reference wilson Finch:
    • Java: import wilson.Finch;
    • Python: from wilson import Finch
  3. Instaniate the Finch class:
    • Java: Finch finch = new Finch();
    • Python: finch = Finch()
  4. Write code to control your Finch:
    • Java: finch.forward(25);
    • Python: finch.forward(25)
  5. Write more code to control your Finch:
    • Java: finch.right(90);
    • Python: finch.right(90)
  6. Run Wilson Java / Python Finch:
    • Both: Replit.com Run Button
    • Note: First time takes longer!
  7. Select your code file to execute:
    • Java: Click the Main link
    • Python: Click the main link
  8. Connect your finch when requested:
    • Both: Click the Connect Finch button
    • Note: Cancel to see only on-screen!
  9. Stop Replit and Add More Code:
    • Both: See the Documentation
    • Note: Lights, leds, sound, ...
  10. Add More Code Files in Root Folder:
    • Both: See the Example "test" Files
    • Note: Even run them at same time!

Do NOT change any other files!

  • Do not change the .replit file
  • Do not change webapp or wilson files
